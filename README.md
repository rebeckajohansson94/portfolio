# Rebecka Johansson – Portfolio

A personal portfolio built with Next.js, TypeScript and Tailwind CSS.

🔗 Live: https://rebecka-portfolio.vercel.app

## Tech Stack

- Next.js 
- React
- TypeScript
- Tailwind CSS 
- Formspree (contact form)

## Getting Started

1. Clone the repository
2. Install dependencies:
   npm install
3. Create a .env.local file in the root directory and add your Formspree form ID:
   NEXT_PUBLIC_FORMSPREE_ID=your_formspree_form_id
4. Run the development server:
   npm run dev
